## Technologies Used
* Frontend: Angular 10
* Backend: Spring Boot, JPA, Java 1.8
* Database: MySQL

## Live Demo
* <a href="#">Demo</a>

## Steps to Run
* Clone this repo.
* Run npm install and npm start on client folder
* Run mvn clean and mvn install on rest folder
* Run the jar created in rest/target folder
* Go to http://localhost:4200
